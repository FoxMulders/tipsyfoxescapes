Chapter 4

Exporting Build-Ready Specifications

The export is the handoff packet for hosts, builders, AV techs, and game masters. It converts selected puzzles into a master blueprint, material lists, setup notes, and operating instructions.

Master Blueprint

The blueprint summarizes story progression, puzzle ordering, dependencies, and reset expectations. Use it as the first page for builders and operators before they inspect individual puzzle cards.

Consolidated Bill of Materials

The bill of materials groups props, electronics, sensors, controllers, fasteners, signage, and printable artifacts across the whole room. For electronic puzzles, inspect parts, wiring diagram lines, build steps, and Arduino-style code before purchasing supplies.

Narrative Justification

Each puzzle should explain why the clue belongs where it appears. This keeps build crews from moving an object that is load-bearing for puzzle logic.

Printing the Packet

  1. Run Export Plan from the Export step.
  2. Download the HTML or markdown export.
  3. Use browser print or your document workflow to save a PDF for build crews.
  4. Keep a digital copy available during game night for quick clue and reset reference.
