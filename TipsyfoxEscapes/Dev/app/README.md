# Escape Room Builder App Scaffold

This folder contains the starter scaffold for MVP development.

## Structure
- `frontend/` - React UI (planning flow screens)
- `backend/` - Express API (generation and session logic)
- `shared/` - shared contracts and types

## Immediate Next Steps
1. Initialize package managers in `frontend` and `backend`.
2. Wire backend endpoints from `Dev/api_contract.md`.
3. Build frontend screens in sprint order from `Dev/sprint_1_execution.md`.
4. Add unit and E2E tests using `QA/mvp_test_plan.md`.
