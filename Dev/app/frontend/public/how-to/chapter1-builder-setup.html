Chapter 1

Initial Setup & Configuration

Start by giving the builder the operational facts it needs: who is playing, where the room runs, which interface you need, and which physical objects the game can use.

Account Sign-In

Create an account or sign in before saving plans. Trial users can explore the builder and export once; paid home or operator plans add save capacity, full catalog access, and additional exports.

Choose a Target Interface

  • Home Party: optimized for a print runbook and a simpler player screen.
  • Commercial Venue: unlocks operator workflows such as the Gamemaster Live Console, player display, staff reset checklists, and multi-display operations.

Set Environment Parameters

Fill out Environment, Players at one time, Total participants, and Session duration. The builder keeps headcount valid: if total participants is lowered below concurrent players, concurrent players automatically clamps to the new total.

Use the environment field as a real space description. Residential users can choose a common room type; commercial users should describe zones, door positions, GM sightlines, and fixed infrastructure.

Balanced Inputs

Keep Players at one time focused on throughput inside the room. Keep Total participants focused on the event total across groups or rotations. The generated puzzle count and parallelism scale from these values.
