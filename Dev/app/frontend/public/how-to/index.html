The Builder Loop

  1. Room: define people, duration, environment, operating mode, and available items.
  2. Theme: pick from generated ideas or provide a custom theme when full-catalog access is available.
  3. Build: generate puzzles that include logic, physical, and electronic variety.
  4. Review: replace only individual puzzle cards that miss the mark.
  5. Export: save or download the plan, then launch live URLs for game night.
Core concept: a planning session is the source of truth. Its session id ties together the builder, export packet, saved plan, Gamemaster Console at /gm/{sessionId}, and player display at /room/{sessionId}/player-display.
